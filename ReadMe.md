# GeeXHP

This project is currently under development and is part of my ongoing thesis work.

# Install with Git (recommended)

Python 3.7 or later is required.

```bash
git clone https://github.com/SarahBarbosa/geexhp.git
cd geexhp
pip install -e .
```
